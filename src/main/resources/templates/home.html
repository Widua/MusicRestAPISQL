/api/all - List all songs

/api/id/{SongID} - Search song by ID

/api/{AuthorName} - Search songs by Author

/api/add - POST method by you can add song to API and Database [ You must as body add JSON that describes song ]

/api/put - PUT method by you can update song [ You must as body add JSON that describes song ]

/api/delete/{songID} - Delete song by ID


/console - H2 Database console, as JDBC URL you must type jdbc:h2:file:./music, there is no Username and no password
